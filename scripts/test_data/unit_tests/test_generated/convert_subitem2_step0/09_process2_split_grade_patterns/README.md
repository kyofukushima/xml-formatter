# 09_process2_split_grade_patterns

## 概要
学年パターン同士の分割処理をテストするケース

## 入力
- `〔第１学年〕` のList要素
- `〔第３学年及び第４学年〕` のList要素
- `〔医療と社会〕` のList要素

## 期待される動作
- 学年パターン同士が同階層として認識され、別々のItem要素に分割される
- 科目名パターンも別々のItem要素に分割される

## 統計期待値
- 学年（1つ記載） -> Item: 1箇所
- 学年（2つ記載） -> Item: 1箇所
- 括弧付き科目名 -> Item: 1箇所
