# テストケース26: Columnが2つあり、1つ目がラベル要素でないList要素の処理

## テスト内容

Columnが2つあるList要素のうち、1つ目のColumnがラベル要素に該当しない場合、変換をスキップしてそのままList要素として残すことを確認します。

## 入力

- 1つ目のList: Column1が「この基準で用いる主な用語及び定義は，ＪＡＳ」（ラベル要素でない）
- 2つ目のList: Column1が「取扱い（ＪＡＳ」（ラベル要素でない）
- 3つ目のList: Column1が「（１）」（ラベル要素）

## 期待値

- 1つ目と2つ目のList要素はそのまま残る（変換されない）
- 3つ目のList要素はItem要素に変換される

## 検証ポイント

- Columnが2つあり、かつ1つ目のColumnがラベル要素に該当しない場合、変換をスキップする
- Columnが3つ以上の場合やTableStructの処理と同様の動作
