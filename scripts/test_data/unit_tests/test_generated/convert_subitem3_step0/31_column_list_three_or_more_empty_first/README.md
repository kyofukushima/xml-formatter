# テストケース31: Columnが3つ以上で最初が空の場合の処理

## テスト内容

Columnが3つ以上あるList要素のうち、1つ目のColumnが空の場合、変換をスキップしてそのままList要素として残すことを確認します。

## 入力

- 1つ目のList: Columnが3つで、1つ目のColumnが空
- 2つ目のList: Columnが2つでラベル要素

## 期待値

- 1つ目のList要素はそのまま残る（変換されない）
- 2つ目のList要素はItem要素に変換される

## 検証ポイント

- Columnが3つ以上で最初のColumnが空の場合、変換をスキップする
- 後続のラベル付きList要素は正常に変換される
