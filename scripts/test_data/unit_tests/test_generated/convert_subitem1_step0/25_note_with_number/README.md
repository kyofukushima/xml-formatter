# 25_note_with_number

## テスト内容

新規ラベルパターン5: 注記 + 数字 (`注記１`形式)

## 期待される動作

- ColumnありList要素で、1つ目のColumnに「注記１」「注記２」などの注記 + 数字のラベルがある
- このListがItem要素に変換される
- 1つ目のColumnの内容（例：「注記１」）がItemTitleに配置される
- 2つ目のColumnの内容がItemSentenceに配置される

## 誤認識防止の検証

以下のパターンは**ラベルとして認識されない**ため、ラベルなし2カラムListとして処理され、List要素のままItem要素に取り込まれる：
- `注記` - 数字なしの注記（ラベルパターンに該当しない）
- `注記（１）` - 注記 + 括弧付き数字（ラベルパターンに該当しない）
- `注記１）` - 注記 + 数字 + ）（ラベルパターンに該当しない）
