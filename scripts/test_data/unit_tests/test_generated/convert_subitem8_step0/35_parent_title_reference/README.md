# 35_parent_title_reference

## テスト内容

親要素（Item要素）のタイトルテキストを参照するロジックをテストするためのテストケースです。

## 期待される動作（convert_item_step0の段階）

- 最初のList「４．２」がItem要素に変換される
- 2つ目のColumnなしListは、1つ目のItem要素に取り込まれる（List要素として）
- 3つ目のList「４．３」は、新しいItem要素に分割される（`convert_item_step0`の段階では親要素参照ロジックは動作しない）

## 検証ポイント

このテストケースは、`convert_item_step0`の段階では直接的に親要素参照ロジックをテストすることはできませんが、
自動生成スクリプトでSubitem1~10のテストケースに変換された際に、親要素参照ロジックが正しく動作することを確認するためのものです。

## 親要素参照ロジックの動作

`convert_subitem2_step0`などの後続ステップでは、以下のロジックが動作します：
- 親要素のタイトルテキストが空の場合、親要素（Item要素）のタイトルテキストを参照する
- 親要素のタイトルテキストを使用してラベル判定が行われる
- 「４．２」と「４．３」は同じラベル種類（ドット区切り数字）なので、同じSubitem2要素に取り込まれる
