# テストケース33: Columnが2つで最初がテキストの場合の分割/取り込み

## テスト内容

Columnが2つで最初がテキスト（ラベルではない）のList要素が複数ある場合の分割/取り込みロジックをテストします。

## 入力

- 1つ目のList: Columnが2つで、1つ目がテキスト（ラベルではない）
- 2つ目のList: Columnが2つで、1つ目がテキスト（ラベルではない）
- 3つ目のList: Columnが2つでラベル要素

## 期待値

- 1つ目のList要素はItem要素に変換される（Sentence要素2つ）
- 2つ目のList要素は1つ目のItem要素に取り込まれる（List要素として）
- 3つ目のList要素は新しいItem要素に変換される（分割）

## 検証ポイント

- Columnが2つで最初がテキストの場合、Sentence要素2つに変換される
- 同じタイプのList要素は取り込まれる
- ラベル付きList要素が出現すると分割される
