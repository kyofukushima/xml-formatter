# 01_process1_branch1_column_list

## テスト内容
ColumnありList（2カラム、ラベル付き）がSubitem2に変換されることをテスト

## 入力
- Subitem1要素内に2カラムのList要素が配置
- 1カラム目にラベル「（１）」、2カラム目に内容「内容1」

## 期待される出力
- List要素がSubitem2要素に変換される
- Subitem2Titleにラベル「（１）」、Subitem2Sentenceに内容「内容1」が配置される

## 仕様書の対応
処理1-分岐1: ColumnありListでラベルがある場合 → Subitem2TitleとSubitem2Sentenceに変換
