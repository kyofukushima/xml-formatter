# 05_alphabet_patterns

## テスト内容

アルファベットパターン（括弧なし）の検証テストです。各パターンは2個ずつ含まれます。

## 含まれるパターン

- **大文字アルファベット**: `A`, `B` (各2個)
- **小文字アルファベット**: `a`, `b` (各2個)
- **全角大文字アルファベット**: `Ａ`, `Ｂ` (各2個)

## 期待される動作

各パターンが別の階層として扱われます。`Subitem2`までに制限されています。

## 階層構造

```
Item Num="1": 大文字アルファベットA
  Item Num="2": 大文字アルファベットB
    └─ Subitem1 Num="1": 小文字アルファベットa
    └─ Subitem1 Num="2": 小文字アルファベットb
        └─ Subitem2 Num="1": 全角大文字アルファベットＡ
        └─ Subitem2 Num="2": 全角大文字アルファベットＢ
```
