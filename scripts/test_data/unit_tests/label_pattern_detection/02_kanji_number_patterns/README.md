# 02_number_patterns

## テスト内容

数字パターン（括弧なし）の検証テストです。

## 含まれるパターン

- **全角数字**: `０`, `１`（各2つ）
- **半角数字**: `1`, `2`（各2つ）
- **漢数字**: `一`, `二`（各2つ）

## 期待される動作

- 全角数字、半角数字、漢数字はそれぞれ別の階層として扱われる

## 階層構造

```
Item Num="1": 全角数字０
Item Num="2": 全角数字１
  └─ Subitem1 Num="1": 半角数字1
  └─ Subitem1 Num="2": 半角数字2
      └─ Subitem2 Num="1": 漢数字一
      └─ Subitem2 Num="2": 漢数字二
```

同じ階層のパターンは同じ階層のタグが振られることを確認します。

## 注意事項

このテストケースは最大7要素程度に簡略化されており、階層が深くなりすぎないようになっています。
