# 09_paren_fullwidth_lowercase_alphabet_patterns

## テスト内容

括弧付き全角小文字アルファベットパターンの検証テストです。各パターンは2個ずつ含まれます。

## 含まれるパターン

- **括弧付き全角小文字アルファベット（半角括弧）**: `(ａ)`, `(ｂ)` (各2個)
- **括弧付き全角小文字アルファベット（全角括弧）**: `（ａ）`, `（ｂ）` (各2個)

## 期待される動作

各パターンが別の階層として扱われます。`Subitem1`までに制限されています。

## 階層構造

```
Item Num="1": 括弧付き全角小文字アルファベット(ａ)
  Item Num="2": 括弧付き全角小文字アルファベット(ｂ)
    └─ Subitem1 Num="1": 括弧付き全角小文字アルファベット（ａ）
    └─ Subitem1 Num="2": 括弧付き全角小文字アルファベット（ｂ）
```
