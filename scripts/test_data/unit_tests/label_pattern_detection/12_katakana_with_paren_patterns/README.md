# 12_katakana_with_paren_patterns

## テスト内容

カタカナ + ）パターンの検証テストです。各パターンは1個ずつ含まれます。

## 含まれるパターン

- **カタカナ + ）**: `カ）`, `キ）`, `ク）`, `ケ）` (各1個)

## 期待される動作

各パターンがItemレベルとして扱われます。カタカナの後に半角閉じカッコ（`）`）が続くパターンが正しく認識されることを確認します。

## 階層構造

```
Item Num="1": カ）
Item Num="2": キ）
Item Num="3": ク）
Item Num="4": ケ）
```

## 注意事項

- カタカナは全角カタカナ（`ア-ヴ`）を対象とします
- 閉じカッコは半角の `）` を使用します
- このパターンは `fullwidth_number_with_paren` や `fullwidth_lowercase_alphabet_with_paren` と同様の優先順位で処理されます
