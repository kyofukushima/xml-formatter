# 04_double_paren_number_patterns

## テスト内容

二重括弧付き数字パターンの検証テストです。

## 含まれるパターン

- **二重括弧付き全角数字**: `（（０））`, `（（１））`（各2つ）
- **二重括弧付き半角数字**: `((1))`, `((2))`（各2つ）
- **二重括弧付き漢数字**: `（（一））`, `（（二））`（各2つ）

## 期待される動作

各パターンが別の階層として扱われます。

## 階層構造

```
Item Num="1": 二重括弧付き全角数字（（０））
Item Num="2": 二重括弧付き全角数字（（１））
  └─ Subitem1 Num="1": 二重括弧付き半角数字((1))
  └─ Subitem1 Num="2": 二重括弧付き半角数字((2))
      └─ Subitem2 Num="1": 二重括弧付き漢数字（（一））
      └─ Subitem2 Num="2": 二重括弧付き漢数字（（二））
```

同じ階層のパターンは同じ階層のタグが振られることを確認します。

## 注意事項

このテストケースは最大7要素程度に簡略化されており、階層が深くなりすぎないようになっています。
