# 01_all_patterns_comprehensive

## テスト内容

すべてのラベルパターンを1つのXMLファイルに含めた包括的なテストケースです。各パターンは2個ずつ含まれます。

**注意**: このテストケースは各パターングループを別々の`Paragraph`要素に分割しています。`Subitem2`までに制限されています。

## 含まれるパターン

### 数字系（括弧なし）
- 全角数字: `０`, `１` (各2個)
- 半角数字: `1`, `2` (各2個)
- 漢数字: `一`, `二` (各2個)

### アルファベット系（括弧なし）
- 大文字アルファベット: `A`, `B` (各2個)
- 小文字アルファベット: `a`, `b` (各2個)
- 全角大文字アルファベット: `Ａ`, `Ｂ` (各2個)

### 括弧付き数字系
- 括弧付き全角数字: `（０）`, `（１）` (各2個)
- 括弧付き半角数字: `(1)`, `(2)` (各2個)
- 括弧付き漢数字: `（一）`, `（二）` (各2個)

### 括弧付きアルファベット系
- 括弧付き大文字アルファベット: `（A）`, `（B）` (各2個)
- 括弧付き小文字アルファベット: `(a)`, `(b)` (各2個)
- 括弧付き全角大文字アルファベット: `（Ａ）`, `（Ｂ）` (各2個)

### 二重括弧付き数字系
- 二重括弧付き全角数字: `（（０））`, `（（１））` (各2個)
- 二重括弧付き半角数字: `((1))`, `((2))` (各2個)
- 二重括弧付き漢数字: `（（一））`, `（（二））` (各2個)

### 二重括弧付きアルファベット系
- 二重括弧付き大文字アルファベット: `（（A））`, `（（B））` (各2個)
- 二重括弧付き小文字アルファベット: `((a))`, `((b))` (各2個)
- 二重括弧付き全角大文字アルファベット: `（（Ａ））`, `（（Ｂ））` (各2個)

### カタカナ系
- カタカナ: `ア`, `イ` (各2個)
- 括弧付きカタカナ: `（ア）`, `（イ）` (各2個)
- 二重括弧付きカタカナ: `（（ア））`, `（（イ））` (各2個)

## 期待される動作

各ラベルパターンが正しく検出され、適切な`LabelPattern`に分類されることを確認します。各パターングループは別々の`Paragraph`要素に分割され、階層は`Subitem2`までに制限されています。
