# 07_double_paren_alphabet_patterns

## テスト内容

二重括弧付きアルファベットパターンの検証テストです。各パターンは2個ずつ含まれます。

## 含まれるパターン

- **二重括弧付き大文字アルファベット**: `（（A））`, `（（B））` (各2個)
- **二重括弧付き小文字アルファベット**: `((a))`, `((b))` (各2個)
- **二重括弧付き全角大文字アルファベット**: `（（Ａ））`, `（（Ｂ））` (各2個)

## 期待される動作

各パターンが別の階層として扱われます。`Subitem2`までに制限されています。

## 階層構造

```
Item Num="1": 二重括弧付き大文字アルファベット（（A））
  Item Num="2": 二重括弧付き大文字アルファベット（（B））
    └─ Subitem1 Num="1": 二重括弧付き小文字アルファベット((a))
    └─ Subitem1 Num="2": 二重括弧付き小文字アルファベット((b))
        └─ Subitem2 Num="1": 二重括弧付き全角大文字アルファベット（（Ａ））
        └─ Subitem2 Num="2": 二重括弧付き全角大文字アルファベット（（Ｂ））
```
