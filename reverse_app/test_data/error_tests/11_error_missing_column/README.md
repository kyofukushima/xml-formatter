# エラーテストケース: Column要素の欠落

## テスト目的
逆変換スクリプトが、複数Column要素がある場合に、Column要素の欠落を検出できるかを検証します。

## テスト内容
- **入力**: ItemTitleがあり、ItemSentenceに3つのColumn要素があるXML
- **期待値**: 4つのColumn要素を持つList要素（ItemTitle + 3つのColumn）
- **実際の出力**: Column要素が1つ欠落している（または順序が異なる）

## 検証項目
1. Column要素の欠落を検出できるか
2. エラーメッセージが適切に出力されるか
3. テストが失敗するか（期待値と実際の出力が一致しない）
