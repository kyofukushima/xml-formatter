# エラーテストケース: Column要素の値変更

## テスト目的
逆変換スクリプトが、複数Column要素がある場合に、Column要素の値変更を検出できるかを検証します。

## テスト内容
- **入力**: ItemSentenceの最初のColumn要素の値が「項目A（変更）」に変更されているXML
- **期待値**: Column要素の値が「項目A」であるList要素
- **実際の出力**: Column要素の値が「項目A（変更）」であるList要素

## 検証項目
1. Column要素の値変更を検出できるか
2. エラーメッセージが適切に出力されるか
3. テストが失敗するか（期待値と実際の出力が一致しない）
